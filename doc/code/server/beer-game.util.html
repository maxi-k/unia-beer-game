apply-transformations

(apply-transformations data transforms)(apply-transformations data transforms default-transform)

Takes a map data and a map transforms that maps from key to a transformative function. For every item in the original map, tries to transform it using the function associated with the same key in the transforms-map. If there is no such key, assumes default-transform, which itselfs defaults to identity.

filter-round-data

(filter-round-data rounds user-role)

Takes a vector of round-data and returns a vector of round-data where the data for reach round only contains the information for the given user-role.

interpose-indexed

(interpose-indexed sep-fn coll)

split-keyword

(split-keyword kw)

Splits a qualified keyword (keyword with a namespace) into a vector of the form [namespace name], with both entries being keywords. If the keyword is not qualified, namespace will be nil.

users-by-event

(users-by-event user-map)

Takes a map from user-id to user-data and reorders it to a map from event-id -> { user-id -> user-data }.
