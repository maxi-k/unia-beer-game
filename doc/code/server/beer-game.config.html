beer-game.config

Various configuration options - defined as a map in definitions - for system-level things like websocket routes and resource paths, as well as domain-level things like the default supply chain or route colors. This namespace is accessible from both the server and the client, and provides the server-side settings as well on the server (from server-config.edn).
